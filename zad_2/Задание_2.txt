Если include ссылается на файл .php:
    Оператор include выполняет поиск файла по названию в папке, где
    расположена программа с этим include.
    Если функция не нашла такого файла, то будет ошибка
    Если функция не смогла открыть его то выведет false
    Если функция нашла этот файл, то есть 2 варианта событий:
        1)Если в файле есть return, то мы присваем переменной
        где используется include значение этого return
        2)Если в файле нет return то возвращаем целочисленное число 1
    То есть если грубо говорить классическая функция, если не смог
    что-то сделать 0, если смог, то 1, если смог и там есть какое-то
    значение то выводит значение.
    include превращает файл который ищем в функцию с определенными встроенными return
    если нашел, но нет у этой файла своего return, то 1, если не нашел 0.
Если файл ссылается на остальные типы например .txt:
    Выводит текст этого файла
    Также имеет свои внутренние return если нашел и открыл 1, если не смог открыть 0
